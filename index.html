CryptWarden

A CLI password manager made with C

Github

What does it do?

1. Manage passwords

CryptWarden’s main purpose is to let users securely store, read, update, and delete passwords.

The CRUD operations can be done by interacting with a menu system.

2. Encryption

Just like how the core of every password managers are encryption, CryptWarden also holds the fort strong.

The security system uses a master password for the manager, which is hashed with PBKDF2 and salted for added security preventing rainbow table attacks.

The user passwords are encrypted with AES-256 CBC using OpenSSL. So sensitive data remains confidential.

3. Logging

CryptWarden keeps a record of all actions.

Logs are easy to view and can be cleared anytime.

BookRanker

A ranked book search engine with C++ 20

Github

What does it do?

1. Add & View Books

BookRanker lets you create your own in-memory library by adding books with details like title, author, and ISBN.

2. Remove Books

Unwanted entries? No problem.

You can remove books by title, author, or ISBN, keeping your library clean and relevant.

3. Ranked Search

The crown jewel of BookRanker is its ranked search system which is smart, fast, and forgiving.

You can search by title or author, even with partial, out-of-order, or slightly misspelled input.

Results are scored and ranked by relevance, with exact matches prioritized for speed.

Just like Google, it even shows how fast your query was processed:
"About 3 results in 64 µs"

4. Smart Design

Built with C++20, BookRanker utilizes clean object-oriented principles and generic design.

Templates and lambda functions reduce repetition, making the code DRY, reusable, and easy to extend.

No menus, no UI, no file storage. Just pure method calls for clarity and focus.

XBank

A simple OOP banking system made with Python

Github

What does it do?

1. Create Accounts

XBank allows users to open Savings or Checking accounts with a name and an initial deposit.

Each account is assigned a unique 10-digit account number starting with 7.

2. Check Balance

Balances are displayed in a neatly formatted table using PrettyTable for better readability.

3. Deposit & Withdraw Funds

Funds can be deposited or withdrawn with proper input validation.

Savings accounts enforce limits on free withdrawals, applying a 1% fee once the limit is exceeded.

4. Transfer Money

Money can be transferred between accounts. Savings accounts apply the same transaction limit and fee rules as with withdrawals.

5. Menu-Driven System

The program runs in a continuous loop with a clear main menu.

Users can navigate through account creation, deposits, withdrawals, transfers, and balance checks without needing additional setup.
